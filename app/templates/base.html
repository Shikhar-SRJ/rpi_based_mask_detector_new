Mask Detector

Here, we are using the Raspberry Pi camera module to perform image processing using OpenCV and CNN.
Although this project is centrally based on face mask detection, this can also be customized as a security camera or can be used in smart homes for access control.
We can extend it to generate alerts when it detects some activity and also add some more CNN models like for object detection or facial expression reading models.
We have used Flask framework to develop this web app and it can further be deployed on the production server using nginx and gunicorn.
